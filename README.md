# resting

school project. made to learn about rest apis, http/https, and php.

## what is it

you can try it live at https://resting-api.mkelvers.tech/ or run it locally.

[api documentation](https://resting-api.mkelvers.tech/docs)

a simple rest api demonstrating:
- http methods (get, post, put, delete)
- json responses
- basic authentication
- routing

## setup

```bash
composer install
cp .env.example .env
docker-compose up -d
```

## dev

```bash
composer dev
```
